The language:

Hints:

  • Any program must begin with the label program followed by the name of the program.
  • After any instruction the ; character must be inserted.
  • In a souce file can contain several programs.

Instructions:

Name Description

read
  • read [variable name];

write, writeln
  • write [variable name];
  • write ["text"];
  • writeln [variable name];
  • writeln ["text"];
if
  • if [condition] begin
        [instructions]
    end;
  • if [condition] begin
        [instructions]
    else begin [instructions]
    end;
